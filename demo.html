buildInstallation & Setup

Prerequisites

Before running this application, install:

  • Node.js (v16.0 or higher) - Download here
  • MongoDB (v4.4 or higher) - Download here
  • npm or yarn (comes with Node.js)

Quick Start

1. Install Server Dependencies:

cd server npm install

2. Install Client Dependencies:

cd client npm install

3. Set Environment Variables:

# Create server/.env MONGODB_URI=mongodb://localhost:27017/testportal JWT_SECRET=your-secret-key OPENAI_API_KEY=your-openai-key # Create client/.env REACT_APP_API_URL=http://localhost:5000/api

4. Start the Application:

# Terminal 1 - Start Server cd server && npm run dev # Terminal 2 - Start Client cd client && npm start
